# dicepro 0.1.0

## Initial CRAN submission

* First release of dicepro
* Semi-supervised deconvolution of bulk RNA-seq data
* Integration of supervised methods (CIBERSORTx, FARDEEP, DCQ, BayesPrism)
* Discovery of unknown cell types using NMF with L-BFGS-B optimization
* Automatic hyperparameter selection via Pareto frontier
* Includes benchmark datasets BlueCode and CellMixtures
